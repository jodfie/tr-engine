Send Debug Report

Help troubleshoot issues by sending system diagnostics to the developer.

Auto-collected diagnostics

  • Browser info (user agent, screen, network type)
  • Audio playback state (if active)
  • Recent browser console errors
  • Server configuration (passwords and API keys are redacted)
  • trunk-recorder config (if available)
  • System health (database, MQTT, streaming status)
  • Audio streaming jitter statistics
  • Recent trunk-recorder warnings and errors (last hour)
  • Server environment (hostname, network interfaces, container detection)

This sends system diagnostics to the tr-engine developer. Secrets (passwords, API keys, tokens) are automatically redacted before transmission.
